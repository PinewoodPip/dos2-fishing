# Installation & Patchnotes

!!! info ""
    [Download here](https://drive.google.com/file/d/1D6Pxq-3wtg3Ggxe0hhio5F8_Os6uOyc0/view?usp=drive_link).
    
    Place the `.pak` within `\Documents\Larian Studios\Divinity Original Sin 2 Definitive Edition\Mods` as with any other mod.

    **[Requires Epip v1075 or newer.](https://www.pinewood.team/epip/)** 
    
    Load after Epic Encounters Core & Origins if you have it; load order does not matter otherwise.

!!! example "Beta!!!"
    This is a beta release of the mod; everything on this site is implemented, but might have room for improvement (ex. balance or progression pace issues).

    We're looking for ALL feedback you might have - we want the mod to be as epic as possible!

    Some things we're looking for feedback on:

    - Minigame difficulty, are some fish too easy/hard?
    - Skill balance, especially Source Infusions
    - Fishermancy progression: does leveling it feel too slow, or too fast?
    - Fish essence power level (though these are mostly placeholder for now, see below)
    - Any fun ideas for more additions to the mod!

    There are also various features we want to implement in future patches:

    - Special effects for fish essences, so they're not just stat boosts - we want to work somewhat like EE Artifacts
    - Different "playstyles" for the different rods (and more varieties of them)
    - Rewards for appraising your collection at the trader (most likely Unique items)
    - More crafting recipes (ex. bait)
    - More trader dialog
    - More fish, especially at the common/uncommon tier to distinguish regions further

??? warning "Limitations"
    - The minigame is currently only available on KB+M; controller support will be added in a later update.
    - Only English text is available for now; localization support will be added in a later update.
    - A Steam Workshop mirror will be released in the future.
